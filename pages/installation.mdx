# Installation & Setup

Here is how you can install our logging package `magik`

Follow these steps.

1. Run `pip install magik`
2. Run `magik init` - a config file will be generated inside `magik_tests/magik_config.json`
3. In the `magik_config.json` file, please add your API key to the `MAGIK_API_KEY` field.

Next, start [logging your data](logging/log_inference_call/langchain).

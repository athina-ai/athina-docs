# Configure Automatic Evals

You can configure evals on [Athina Dashboard](https://app.athina.ai/evals/config) to run automatically against [logged inferences](/logging).

The results are visible in the **Evals** tab on the Inference detail (`/observe/inference/:inference_id`) page.

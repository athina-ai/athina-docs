If you'd like to access your Athina logs and evaluation results programmatically, you can use our GraphQL API.

GraphQL provides a flexible and powerful way to query inference logs and evaluation data.
If you want to know more about graphql, you can visit the official [graphql documentation](https://graphql.org/learn/).
